SpanDrive is a file repository system built with React and Firebase. This project allows users to create folders, upload files, and manage their files securely, making it an efficient solution for file organization and storage.
Features: - **File & Folder Management**:
  - Create folders and subfolders.
  - Upload files inside folders.
  - Navigate through nested folder structures.
  
- **User Authentication**:
  - Separate login credentials for admins and regular users.
  - Admins have full access to all files and folders.

- **File Security**:
  - Encryption ensures that files and folders are accessible only to authorized users.

- **Responsive Design**:
  - Fully responsive user interface for seamless access on different devices.

 ##Technologies Used

- **Frontend**:
  - React
  - Material-UI (Version 4)
  - TailwindCSS

- **Backend**:
  - Firebase Authentication
  - Firebase Firestore (Database)
  - Firebase Storage
